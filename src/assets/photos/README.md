# Fotos predeterminadas

Deja aquí imágenes (.webp, .png, .jpg) y aparecerán solas en los widgets
y en la app Fotos. No hay que registrarlas en ningún sitio: Vite las descubre
en tiempo de compilación.

Pásalas por `node scripts/build-icons.mjs`? No: ese script es solo para iconos.
Las fotos se sirven tal cual, así que conviene subirlas ya optimizadas.
